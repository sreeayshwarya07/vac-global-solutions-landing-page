# VAC Global Solutions Landing Page

## Overview
This project is a single-page corporate landing page developed using Astro. It is designed to represent VAC Global Solutions and showcase its services with a clean and modern interface.

## Features
- Responsive design for mobile and desktop
- Minimal and consistent UI
- Fast loading performance
- Component-based structure

## Sections
- Hero
- About Us
- Services Overview
- Why Choose Us
- Contact Form
- Footer

## Tech Stack
- Astro
- HTML
- CSS

## Run Locally
npm install
npm run dev
